# MONTHLY Finance Tracker

## Install on Your Phone

### Android (Chrome/Edge/Samsung Internet):
1. Open the app in your browser
2. Tap the menu (⋮) 
3. Select "Add to Home screen" or "Install app"
4. Tap "Add" or "Install"
5. The app icon will appear on your home screen

### iPhone/iPad (Safari):
1. Open the app in Safari
2. Tap the Share button (□↑)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add"
5. The app icon will appear on your home screen

## Features
- Works offline after first load
- All data stored locally on your device
- No internet connection required
- Acts like a native app

## Hosting Options

### Option 1: GitHub Pages (Free)
1. Create a GitHub account
2. Create a new repository
3. Upload all files
4. Enable GitHub Pages in settings
5. Access via: `https://yourusername.github.io/repo-name`

### Option 2: Netlify/Vercel (Free)
1. Create account on Netlify or Vercel
2. Drag and drop the folder
3. Get instant URL
4. Access from anywhere

### Option 3: Local Network
1. Keep the Python server running: `python3 -m http.server 8000`
2. Find your computer's IP address: `ip addr` or `ifconfig`
3. On phone, open: `http://YOUR_IP:8000`
4. Add to home screen
